Online Hash
OnlineHash.xyz is an 100% in-browser hashing tool. Nothing is sent to a remote server - its all done on your computer using the awesome crypto.js library.

Why?
We all use online hashing tools from time to time, occasionally for sensitive data that we really shouldn't be pasting into the first site we find for "online sha hash" or "online md5 hash" searched. Sure we could do it on the command line, but we're lazy.
